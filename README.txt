İNfüzyon Hesaplayıcı - iPhone App Gibi Kullanım

2) İçindeki tüm dosyaları GitHub Pages, Netlify Drop veya herhangi bir statik site servisine yükle.

4) Paylaş > Ana Ekrana Ekle.
